⬢ *SISTEMA DE ESTACIONAMENTO*

---



*Controle simples e eficiente de vagas*

Projeto desenvolvido com JavaScript, HTML e CSS para simular um sistema de estacionamento.
A aplicação permite gerenciar veículos de forma prática, registrando entrada, saída e cálculo de tempo de permanência.


---

⬡ *Propósito*

Simular um sistema real de estacionamento

Praticar manipulação do DOM com JavaScript

Trabalhar lógica de programação aplicada

Desenvolver interface simples e funcional

Integrar HTML, CSS e JS em um projeto completo



---

⬡ *Funcionalidades*

Registro de veículos na entrada

Remoção de veículos na saída

Cálculo de tempo de permanência

Lista dinâmica de carros estacionados

Interface responsiva e intuitiva

Atualização em tempo real
